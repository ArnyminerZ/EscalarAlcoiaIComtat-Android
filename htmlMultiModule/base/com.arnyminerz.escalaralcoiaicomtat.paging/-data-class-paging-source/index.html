The paging source for loading the data for the Paging 3 library. Loads DataClasses content. If all the IDs are null, the areas will be loaded. Order of preference in case of multiple non-null parameters are specified is:

  1. sectorId

  2. zoneId

  3. sectorId

Author

Arnau Mora

Since

20210819

Parameters

searchSession

The session for loading the data from.

areaId

The ID of the area to load. Will load zone sectors.

zoneId

The ID of the zone to load. Will load child sectors.

sectorId

The ID of the sector to load. Will load child paths.
