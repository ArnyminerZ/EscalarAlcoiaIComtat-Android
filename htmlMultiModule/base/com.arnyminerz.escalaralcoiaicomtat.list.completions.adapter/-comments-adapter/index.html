CommentsAdapter

class CommentsAdapter(activity: CommentsActivity, items: ArrayList<MarkedDataInt>) : RecyclerView.Adapter<CommentsViewHolder>
Content copied to clipboard

The RecyclerView adapter for showing the comments that people have published into a Path. This is meant to be used inside CommentsActivity. It's assumed that items contain valid comments (with content), otherwise, NullPointerException may be thrown.

Arnau Mora

activity

The MarkedDataInt elements to show.
